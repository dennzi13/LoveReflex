--------------------------------------------------------

・お触りすると時間が経過していきます。時間切れになると自動で終了します。
・ゲージが満タンの状態で刺激を受けると射精してしまいます。隣は射精できる回数です。射精できる回数が0になっても自動で終了します。
・ゴムえっちは3回まで行えます。
・\1側をダブルクリックすることでオプションメニューを開くことができます。

●触り反応
・\0側
　頭つつき：反応なし
　頭なで：頭なで
　耳つつき：耳つつき
　耳なで：耳なで
　口なで：反応なし
　口つつき：キス
　口ころ：フェラ
　胸なで：胸なで
　胸つつき：胸触り
　胸ころ：パイズリ
　手なで：反応なし
　手つつき：手コキ
　お腹つつき：反応なし
　お腹なで：お腹なで
　xxxなで：反応なし
　xxxつつかれ：セックス
　ふとももなで：ふとももなで
　ふとももつつき：反応なし
　ふとももころ：ふとももコキ

・\1側
　ゲージなで/つつき：オナニー
　ゲージ以外の箇所つつき：オプションメニュー表示
